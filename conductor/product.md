# Initial Concept\n\n建立一個極簡的 Bash 腳本應用，透過 Wireless ADB 連線手機，並每 5 秒自動在指定座標執行滑動動作，不需額外安裝 Python 或其他語言環境。
\n## Product Guide - AutoSwipe Bash\n\n### 產品願景\n為 Linux/WSL2 使用者提供一個極簡、無依賴的自動化工具，透過 Wireless ADB 執行定時的螢幕滑動操作。\n\n### 目標使用者\n- 需要在 WSL2 環境下快速自動化手機操作的使用者。\n- 偏好使用原生系統工具（Bash, ADB）而非安裝大型開發環境的人。\n\n### 核心功能\n1. **互動式連線管理**：\n   - 啟動時引導使用者進行 Wireless ADB 配對（輸入 IP:Port 與配對碼）。\n   - 自動檢測連線狀態。\n2. **自動化滑動邏輯**：\n   - 從 `.env` 讀取起始座標 `(x1, y1)` 與結束座標 `(x2, y2)`。\n   - 每隔指定秒數執行一次滑動。\n   - 支援設定總執行時長，時間到自動停止並中斷 ADB 連線。\n3. **即時回饋**：\n   - 終端機顯示當前已滑動次數與剩餘執行時間。\n   - 顯示 ADB 原始輸出以便排錯。\n\n### 技術規範\n- **語言**：Bash Script\n- **依賴**：Android SDK Platform-Tools (adb)\n- **設定格式**：`.env` 檔案
