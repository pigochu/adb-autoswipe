## 運作環境

在 WSL2 上經過 Ubuntu 24 測試沒問題，理論上只要 Linux 系統可以安裝 adb 都可以跑。

## 前置作業

安裝 adb

```bash
sudo apt install adb
```

## 使用方式

**手機端:**

1. 啟用開發者模式，請自行查詢您的品牌啟用方式，一般來說
2. 於開發者模式設定啟用 `無限偵錯`
3. 於 `無限偵錯` 設定畫面內使用配對碼配對裝置。
4. 要記住配對的 IP : PORT , 配對碼
5. 要記住連線的 IP : PORT (通常是主畫面)
